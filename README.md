# Alan's Blog

这是我的个人技术博客。

## 访问博客

博客地址：**https://alanniew.github.io/**

## 本地运行

```bash
bundle install
bundle exec jekyll serve
```

然后访问 http://localhost:4000

## 项目结构

```
├── _config.yml       # Jekyll 配置
├── _layouts/         # 布局模板
├── _posts/          # 博客文章
├── assets/          # 静态资源
├── index.html       # 首页
└── about.md         # 关于页面
```

## 技术栈

- Jekyll - 静态站点生成器
- GitHub Pages - 网站托管
- Markdown - 内容编写
