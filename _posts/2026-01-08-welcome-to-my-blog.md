## 关于这个博客

这个博客托管在 GitHub Pages 上，使用 Jekyll 静态网站生成器构建。它具有以下特点：

- **简洁美观** - 采用现代简约设计风格
- **响应式布局** - 适配各种设备屏幕
- **快速加载** - 纯静态网站，访问速度快
- **易于维护** - 使用 Markdown 编写文章

## 我会分享什么

在这个博客中，我会分享：

1. **技术学习笔记** - 记录学习新技术的过程和心得
2. **项目经验** - 分享项目开发中的问题和解决方案
3. **工具推荐** - 推荐好用的开发工具和资源
4. **随笔思考** - 对技术行业的一些思考

## 如何写新文章

在 `_posts` 目录下创建新的 Markdown 文件，文件名格式为：

```
年份-月份-日期-文章标题.md
```

文章开头需要添加 Front Matter：

```yaml
---
layout: post
title: 文章标题
date: 2026-01-08 10:00:00 +0800
categories: [分类1, 分类2]
---
```
